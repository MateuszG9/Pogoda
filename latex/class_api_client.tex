\doxysubsection{Detailed Description}
Klasa odpowiedzialna za komunikacje z API GIOS. 

Klasa wykorzystuje QNetwork\+Access\+Manager do wysyłania żądań HTTP i zwraca dane w postaci obiektów QFuture dla asynchronicznego przetwarzania.
